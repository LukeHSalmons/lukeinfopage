Password Hacked Checker - Python

This project was also built using Python and is used to check if your password has been leaked using the 'Have I Been Pwned' API. The program is able to read all passwords from a text file, hash them and retreive a list of hash passwords from the website that match the first few digits of your password. The program then compares these hashes on your local machine to your hashed password. The program works this way so the website never sees your full hashed password and your passwords remain secure.
